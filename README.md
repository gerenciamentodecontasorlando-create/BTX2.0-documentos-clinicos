# BTX2.0-documentos-clinicos
Documentos Clínicos em PDF é uma ferramenta simples para geração de fichas clínicas, receituários, orçamentos, atestados e laudos em PDF, voltada a dentistas, fonoaudiólogos, fisioterapeutas e outros profissionais da saúde. Não é prontuário eletrônico. O armazenamento dos documentos é de responsabilidade do profissional
